# MIDNIGHT — 真夜中の刀

A scroll-driven WebGL experience. One katana, one cut, one legend.

Real-time three.js rendering — no video, no pre-rendered frames. Scrolling drives a damped cinematic choreography: the blade wakes, the stand dissolves to sand, blue fire ignites along the hamon, and a single cut plants the sword into volcanic rock.

## Run locally

```
node server.js
```

Then open http://localhost:8765.

## Stack

- three.js (WebGL, custom GLSL: fire, embers, sand dissolve, shockwave)
- GSAP + ScrollTrigger + Lenis (damped scroll choreography)
- Models authored in Blender, HDRI baked from the scene itself
